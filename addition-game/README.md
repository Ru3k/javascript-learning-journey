# ➕ Addition Game

A simple **Addition Game** built using **HTML, CSS, and JavaScript**.
The game generates two random numbers and asks the user to calculate their sum.

## 🚀 Features

* Generates **random numbers** every time the game starts.
* Users can **enter their answer** in the input field.
* Displays a **success message** if the answer is correct.
* Displays a **try again message** if the answer is incorrect.
* **Restart button** generates new numbers and clears the previous result.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap

## 🎮 How to Play

1. Two random numbers will appear on the screen.
2. Add the numbers.
3. Enter the result in the input field.
4. Click the **Check** button to verify your answer.
5. Click **Restart** to generate new numbers.

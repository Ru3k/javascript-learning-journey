# 🎨 Background Color Changer

A simple **JavaScript mini project** that allows users to change the background color of the page by clicking different color buttons.

This project demonstrates basic **DOM manipulation** using JavaScript.

---

## 📸 Project Preview

When a user clicks on a color button, the background color of the page changes and the selected color code is displayed on the screen.

---

## 🚀 Features

- Change background color with a single click
- Display the selected color code dynamically
- Clean and responsive UI
- Beginner-friendly JavaScript logic

## ⚙️ How It Works

1. The page contains **four color buttons**.
2. Each button triggers a **JavaScript function** when clicked.
3. The function:
   - Changes the **background color** of the page.
   - Updates the **color code text** displayed on the screen.

Example:

```javascript
document.getElementById("background-color-page").style.backgroundColor = "#6fcf97";
```

This line dynamically updates the background color.

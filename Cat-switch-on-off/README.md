# 🐱 Cat & Bulb Toggle Mini Project

A simple **JavaScript DOM manipulation project** where clicking **ON / OFF buttons** controls a bulb and a cat image.

This project demonstrates how JavaScript can dynamically update images, text, and styles based on user interaction.

---

## 🚀 Live Behavior

* Clicking **ON**

  * Bulb turns **ON**
  * Cat appears normally
  * Status shows **"Switched On"**

* Clicking **OFF**

  * Bulb turns **OFF**
  * Cat eyes glow in the dark
  * Status shows **"Switched Off"**

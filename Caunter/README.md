# Counter App 🔢

A simple **JavaScript Counter Application** that allows users to increase, decrease, and reset a counter value.
This mini project helped me practice **DOM manipulation and event handling in JavaScript**.

---

## 🚀 Features

* Increase the counter value
* Decrease the counter value
* Reset the counter to **0**
* Dynamic color change based on value:

  * 🟢 **Green** when the value is positive
  * 🔴 **Red** when the value is negative
  * ⚫ **Black** when the value is zero

---

* **HTML5** – Structure of the app

## ⚙️ How It Works

1. The counter value is stored in the DOM using `textContent`.
2. JavaScript functions update the value when buttons are clicked:

   * `onIncrement()` increases the number
   * `onDecrement()` decreases the number
   * `onReset()` sets the value back to **0**
3. The color of the counter changes dynamically depending on the value.

# 🚀 Subscribe to Us Form

A simple and responsive **Subscribe Form** built using HTML, CSS, and JavaScript with basic form validation.

---

## 📌 Project Overview

This project demonstrates how to:

* Create a structured HTML form
* Apply styling using CSS & Bootstrap
* Handle user input using JavaScript
* Perform basic validation using the `blur` event

---

## 🎯 Features

✅ Clean and responsive UI
✅ Input validation on focus loss (blur event)
✅ Error messages for empty fields
✅ Styled using Bootstrap and custom CSS

## ⚙️ How It Works

* When the user clicks outside (blur) the input field:

  * If the field is empty → shows `Required*`
  * If the field has value → clears error message

---

## ▶️ How to Run

1. Download or clone the repository
2. Make sure all files are in the same folder:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open `index.html` in your browser

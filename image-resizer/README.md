# 🖼️ Image Resizer App

A simple **JavaScript mini project** that allows users to dynamically **increase or decrease the width of an image** using buttons.

This project demonstrates **DOM manipulation, event handling, and basic UI interaction using JavaScript**.

---

## 🚀 Features

* ➕ Increase image width by **5px**
* ➖ Decrease image width by **5px**
* ⚠️ Displays warning messages when limits are reached
* 📏 Shows the **current width of the image**
* 📱 Responsive layout using **Bootstrap**

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **Bootstrap** – Responsive design
* **JavaScript (Vanilla JS)** – DOM manipulation and logic

## ⚙️ How It Works

1. The image starts with a **default width of 200px**.
2. Clicking the **"+" button** increases the width by **5px**.
3. Clicking the **"-" button** decreases the width by **5px**.
4. The width is limited between:

* **Minimum:** 100px
* **Maximum:** 300px

If the user tries to exceed these limits, a **warning message** is displayed.

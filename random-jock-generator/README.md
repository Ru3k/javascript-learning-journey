# 😂 Random Joke Generator

A simple and interactive web app that fetches and displays random jokes using an API. Built using **HTML, CSS, Bootstrap, and JavaScript (Fetch API)**.

---

## 🚀 Features

* 🎯 Fetches a new random joke on button click
* ⏳ Displays a loading spinner while fetching data
* ❌ Handles API errors gracefully
* 📱 Responsive UI using Bootstrap
* 🎨 Clean and minimal design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **Bootstrap 4** – Layout & responsiveness
* **JavaScript (ES6)** – Logic & API handling
* **Fetch API** – Data fetching

---

## ⚙️ How It Works

1. User clicks the **"Joke"** button

2. A loading spinner appears

3. A request is sent to the API:

   https://apis.ccbp.in/jokes/random

4. The joke is fetched and displayed on the screen

5. Spinner disappears after loading

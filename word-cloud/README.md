# 🌥️ Word Cloud Generator

A simple and interactive **Word Cloud Generator** built using **HTML, CSS, Bootstrap, and JavaScript**.
Users can dynamically add words, and each word is displayed with a random font size to create a cloud-like visual effect.

---

## 🚀 Features

* ✨ Dynamic word generation
* 🎨 Random font sizes for each word
* ⚡ Instant UI updates (no page reload)
* ❌ Input validation (prevents empty entries)
* 🧩 Clean and responsive layout using Bootstrap

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Bootstrap 4**
* **JavaScript (DOM Manipulation)**

## 🧠 How It Works

1. Predefined words are loaded on page start.
2. Each word is displayed with a **random font size**.
3. User can:

   * Enter a new word
   * Click the **Add** button
4. The new word is instantly added to the cloud.
